Detailed Description

This is the main file of the fidi (φίδι) lint tool. Here we implement commands line parsing, and then either put out a help screen or create a parser and check the input.

Main function.

Implement command line parsing. –help or –version are terminal options, in the sense that we print out the help text or version number and then exit. Otherwise, first, create a parser. Secondly, determine if a filename has been provided, or we should read from ther standard input; and then pass the appropriate input stream to the parser, and then perform the syntax and sanity checks. Finally, it generates a graph description in dot(1) format, which can then be processed to create a graph of the request and the cascade of the resulting requests that it generates, to provide a visual depiction of the requested behaviour.
