a class that compares struct EdgeDetails

This class has a single method that can help order a set of EdgeDetails structure instances based on the sequence number.

compare two struct EdgeDetails

This method compares two struct EdgeDetails based on their sequence numbers, and weakly orders them in reverse sequence number order. The lower numbered sequences come first in the list.
